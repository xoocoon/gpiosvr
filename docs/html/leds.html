leds executable¶

Executable script for blinking LEDs in specific patterns.

Arguments¶

The required positional arguments specify the blink pattern and GPIO(s):

  1. Total number of blink cycle(s) for the specified LED(s).

  2. Millisecond duration of the on phase of the LED(s), as well as the following off phase.

  3. After the blink cycles, the millisecond duration for which the LED(s) shall remain on. 0 if “forever”.

  4. Up to three GPIO numbers for the LEDs to turn on and off in each blink cycle.

The standard arguments from gpiosvr.shell.GpioControlShell.addStandardArgs() are accepted.

For connecting to a specific GPIO server, the arguments --strategy and possibly --socket are required. As an alternative to these arguments, the environment variables GPIO_STRATEGY and GPIO_SOCKET_PATH can be used, respectively. If neither these arguments nor environment variables are given, the standard strategy of GpioControlShell is applied.
