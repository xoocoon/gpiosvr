class gpiosvr.uart.UartParameters(channel: int, uartParameterConfig: dict)

Properties describing the UART parameters for a specific UART channel.

__init__(channel: int, uartParameterConfig: dict)
Parameters:

  • channel¶ (int) – The UART channel for which the parameters are valid.

  • uartParameterConfig¶ (dict) –

    Under the key baud (str), the dict must contain an int value. It determines the baud rate of the UART connection.

    Under the key dataBitCount (str), the dict may contain an int value. It determines the number of data bits in the UART communication. Defaults to 8.

    Under the key stopBitCount (str), the dict may contain an int value. It determines the number of stop bits in the UART communication. Defaults to 1.

    Under the key parity (str), the dict may contain a parity value listed in Uart. It determines the parity type in the UART communication. Defaults to PARITY_NONE.

    Under the key isCloseOnExit (str), the dict may contain a bool value. If True, the UART connection will be closed as soon as the opening process is terminating. If False, the UART connection will stay open. Defaults to True.

    Under the key uartPauseBeforeReading_ms (str), the dict may contain an int value. It determines the pause in milliseconds after a command was sent via the UART channel, before reading any bytes in response. Defaults to the value of UART_PAUSE_BEFORE_READING_DEFAULT_MS.

    Under the key uartReadTimeout_ms (str), the dict may contain an int value. When trying to read bytes from the UART channel, the value determines the maximum time span in milliseconds to wait. If no byte was received within that time span, reading will be aborted. Defaults to the value of UART_READ_TIMEOUT_DEFAULT_MS.

    Under the key uartReadBufferBytes (str), the dict may contain an int value. When reading bytes from the UART channel, the value determines the maximum number of bytes to read in one go. Defaults to the value of UART_READ_BUFFER_BYTES_DEFAULT.

    Under the key uartReadDelimiter (str), the dict may contain a str, bytes or a bytearray. When reading bytes from an UART channel, the byte sequence serves as a delimiter. A str prefixed with 0x` is interpreted as a hexadecimal value. Defaults to the value of UART_READ_DELIMITER_DEFAULT.

    Under the key uartReadDelimiterMinCount (str), the dict may contain an int value. When reading bytes from an UART channel, the the value determines the minimum number of occurrences of the delimiter. That is, if the delimiter specified under the key uartReadBufferBytes occurs at least that many times anywhere in the byte sequence read so far, reading will be stopped. Any surplus bytes will be discarded. Defaults to the value of UART_READ_DELIMITER_MIN_COUNT_DEFAULT.

Raises:

ValueError if a value in the configuration is outside the allowed range.
